# Quicknote

Quicknote is a polished, Google Keep-inspired notes web app built with React,
TypeScript, and Vite.

## Features in this version

- Responsive Keep-like layout with a sticky header, sidebar, search, and profile controls.
- Note composer with title, body, and color selection.
- Pinned and unpinned note sections.
- Label chips and search filtering across note titles, content, and labels.
- Modern card UI with soft shadows, warm colors, and mobile-friendly breakpoints.

## Roadmap

1. Add local persistence so notes survive browser refreshes.
2. Support editing, deleting, archiving, and restoring notes.
3. Add custom labels and reminders.
4. Introduce authentication and cloud sync.
5. Add drag-and-drop card ordering and richer note types.

## Development

Install dependencies:

```bash
npm install
```

Start the local development server:

```bash
npm run dev
```

Create a production build:

```bash
npm run build
```
